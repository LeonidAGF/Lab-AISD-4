# **Лабораторная работа №4**

### **Тема**: 

Сбалансированные деревья

#### ***Что нужно былос сделать*:** 

Построить упорядоченное дерево для хранения и поиска элементов с  уникальными строковыми ключами и вещественными значениями, где ключ –  непустая последовательность латинских букв не более чем из 6 символов.  Ключи упорядочиваются лексикографически. 

#### **Вариант:** 

Красно-черное дерево 

**Для каждого вида дерева реализовать следующие 4 операции:** 

1. Добавление узла к дереву по ключу  (с балансировкой при  необходимости). 
2. Удаление узла из дерева по ключу ключу  (с балансировкой при  необходимости)  
3.  Печать дерева 
4.  Поиск значения элемента по заданному ключу.

Для красно-черного дерева предусмотренна при выводе на печать маркировка  цвета узла разными видами скобок.

Предусмотренна возможность ввода команд из текстового файла.

Созданно 5 юнит тестов.
