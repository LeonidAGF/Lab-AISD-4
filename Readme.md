# **Лабораторная работа №4**

### **Тема**: 

Сбалансированные деревья

#### ***Что нужно было сделать*:** 

Построить упорядоченное дерево для хранения и поиска элементов с  уникальными строковыми ключами и вещественными значениями, где ключ –  непустая последовательность латинских букв не более чем из 6 символов.  Ключи упорядочиваются лексикографически. 

#### **Вариант:** 

Красно-черное дерево 

**Для дерева реализованы следующие 4 операции:** 

1. Добавление узла к дереву по ключу. 
2. Удаление узла из дерева по ключу ключу  
3.  Печать дерева 
4.  Поиск значения элемента по заданному ключу.

Для красно-черного дерева предусмотрена при выводе на печать маркировка  цвета узла разными видами скобок: () - чёрный,  [] - красный.

Предусмотрена возможность ввода команд из текстового файла.

**<u>Создано 5 юнит тестов.</u>**
